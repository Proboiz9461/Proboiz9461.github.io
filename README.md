# Shaurya's Creations

Now structured as **100 standalone apps**:

- 50 tool apps in `apps/tools/tool-XX/`
- 50 game apps in `apps/games/game-XX/`
- Every app has 3 files: `index.html`, `style.css`, `script.js`
- Total generated app files: **300**

Main `index.html` is a searchable launcher for Shaurya's Creations that opens each standalone app page.

## Run locally

```bash
python3 -m http.server 8000
```
